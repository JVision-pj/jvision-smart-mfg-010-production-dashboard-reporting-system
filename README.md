# 生產報表看板管理艙

彙整現場生產數據並以視覺化儀表板呈現，供各層級即時掌握生產狀況。

- 原始系統：生產報表看板系統（Production Dashboard/Reporting System）
- 產業分類：智慧製造｜製造管理
- 使用部門：生產、生管、廠務主管、高階經營層
- 工作流程：數據採集 → 資料整合清洗 → 儀表板呈現 → 異常警示推播 → 決策應用
- AI 應用：AI自動生成生產異常摘要報告、智慧預測產量達成趨勢
- 主要 KPI：報表產出時效、異常反應時間、數據準確率
- 代表產品：Power BI、Tableau、AVEVA/Wonderware System Platform Dashboard
